📌 Project Overview

This project is a PHP & MySQL Blog Application developed as part of the ApexPlanet Software Pvt. Ltd. Web Development Internship.

The application allows users to register, log in securely, and manage blog posts using complete CRUD (Create, Read, Update, Delete) functionality. It also includes search, pagination, role-based access control, and several security enhancements.

---

🚀 Features

User Authentication

- User Registration
- User Login
- Secure Password Hashing
- Session Management
- Logout Functionality

CRUD Operations

- Create Blog Posts
- View Blog Posts
- Edit Existing Posts
- Delete Posts (Admin Only)

Search & Pagination

- Search posts by title or content
- Pagination for easier navigation

Security Enhancements

- Prepared Statements (SQL Injection Prevention)
- Password Hashing
- Server-side Validation
- Client-side Validation
- Role-Based Access Control (Admin & Editor)

User Interface

- Responsive design using Bootstrap
- Custom CSS Styling
- Clean and Simple Layout

⚙ Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project into the "htdocs" folder.
4. Create a MySQL database named "blog".
5. Import the required tables.
6. Update database credentials in "config/db.php".
7. Open your browser and visit:

http://localhost/blog_app
